### Workout Planner (Version 1)

A very simple **terminal-based Workout Planner in Python** that uses the **Google Gemini API** to generate a weekly workout plan.

This is intentionally minimal and beginner-friendly:

- **Single Python file**: `main.py`
- **Plain Python** (no frameworks, no classes)
- Runs in the **terminal/command line**

---

### 1. Prerequisites

- **Python** 3.9 or later
- A **Gemini API key** from Google
  - You can create one at: `https://aistudio.google.com/app/apikey`
- Internet connection (the plan is generated by the online Gemini API)

> Note: Google now recommends a newer library called `google-genai`, but this project
> deliberately uses the older `google-generativeai` package to match these instructions.

---

### 2. Install dependencies

Open a terminal in the `workout-planner` folder, then run:

```bash
pip install -r requirements.txt
```

This installs the **google-generativeai** library.

---

### 3. Set your Gemini API key

The program reads your API key from an **environment variable** called `GEMINI_API_KEY`.

Replace `"YOUR_API_KEY_HERE"` with your actual key.

**On Windows (PowerShell):**

```powershell
$env:GEMINI_API_KEY="YOUR_API_KEY_HERE"
```

**On macOS / Linux (bash / zsh):**

```bash
export GEMINI_API_KEY="YOUR_API_KEY_HERE"
```

You will need to do this in each new terminal session unless you add it to your shell profile.

---

### 4. Run the program

From inside the `workout-planner` folder, run:

```bash
python main.py
```

You will be asked a few questions:

- Your **fitness goal** (e.g. strength, weight loss, endurance)
- **Workout days per week**
- Your **experience level** (beginner or intermediate)

The program sends this information to Gemini and prints a simple weekly workout plan.

---

### 5. Example terminal session

Below is an example of what running the program might look like
(`>>>` just shows what the user types):

```text
=== Simple Workout Planner (Gemini Version 1) ===

What is your main fitness goal? (e.g. strength, weight loss, endurance): >>> strength
How many days per week can you work out? (1-7): >>> 3
What is your experience level? (beginner / intermediate): >>> beginner

=== Your Weekly Workout Plan ===

Day 1 – Full Body Strength
- Squats: 3 sets x 8–10 reps
- Push-ups (knees if needed): 3 sets x 8–10 reps
- Bent-over dumbbell rows: 3 sets x 8–10 reps
- Glute bridges: 3 sets x 12 reps
- Plank: 3 sets x 20–30 seconds

Day 2 – Upper Body
- Dumbbell bench press: 3 sets x 8–10 reps
- One-arm dumbbell rows: 3 sets x 8–10 reps
- Dumbbell shoulder press: 3 sets x 8–10 reps
- Biceps curls: 3 sets x 10–12 reps
- Triceps dips (bench): 3 sets x 8–10 reps

Day 3 – Lower Body & Core
- Goblet squats: 3 sets x 10–12 reps
- Romanian deadlifts: 3 sets x 8–10 reps
- Lunges: 3 sets x 8–10 reps per leg
- Calf raises: 3 sets x 15 reps
- Side plank: 3 sets x 20–30 seconds per side

Remember: warm up 5–10 minutes before each session and stretch at the end.

=== End of Plan ===
```

Your exact output will be different, because it is generated by the Gemini model.

---

### 6. File overview

- **`main.py`**  
  Handles:
  - asking for user input,
  - building a text prompt,
  - calling the Gemini API,
  - printing the workout plan.

- **`requirements.txt`**  
  Contains the Python dependency:
  - `google-generativeai`

- **`README.md`**  
  The file you are reading now. Explains how to:
  - install dependencies,
  - set the API key,
  - run the script.
